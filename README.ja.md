# webrtc-test
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

これはSkyWay JavaScript SDKを使用して構築された、シンプルなWebRTCベースのビデオチャットアプリケーションです。

## デモ
アプリケーションは[SkyWay チュートリアルページ](https://webrtc.ecl.ntt.com/en/js-tutorial.html)で試すことができます。

## 機能
- ピアからのリアルタイムのビデオおよびオーディオ通信
- 音声およびビデオ入力デバイスの選択機能
- 簡単なユーザーインターフェース

## 要件
- WebRTCをサポートする最新のウェブブラウザ

## 使い方
1. `index.html` ファイルをWebブラウザで開きます。
2. ブラウザにカメラとマイクへのアクセス権限を許可します。
3. 他の人があなたに電話をかけられるように、ユーザーIDを共有します。
4. 通話したい相手のユーザーIDを入力し、「通話」ボタンをクリックします。
